# LIMEN AI Edge-Case Atlas

This project serves as a comprehensive repository for documenting and analyzing edge cases in AI systems, with a focus on rare languages, underrepresented jurisdictions, and non-English governance evidence. The atlas aims to provide structured evidence for academic papers, policy briefs, and thesis chapters related to AI governance, digital identity, and trust architectures.

## Project Structure

- `journal.md`: Daily entries documenting progress, findings, and next steps.
- `sources/sources.md`: Catalog of sources used in the research.
- `methods.md`: Description of methodologies and algorithms applied.
- `claims.md`: Research claims and their supporting evidence.
- `data/README.md`: Data structure and codebook.
- `results/`: Output artifacts from analyses.
- `draft/preprint.md`: Draft manuscript for publication.
- `manifest.json`: Metadata and configuration for the project.
- `publication-goal-card-current.md`: Current publication sprint objectives (if applicable).
