# LIMEN AI Edge Case Atlas Dashboard and Paper Forge

## Project Overview
This project focuses on transforming LIMEN data structures into a comprehensive dashboard and research publications. The primary objectives include:

- Designing an interactive dashboard for LIMEN data visualization
- Forging academic papers and articles based on the analysis
- Establishing a structured framework for future research and publication

## Project Structure
Key directories and files:

- `journal.md`: Research log and progress tracking
- `dashboard/`: Dashboard specifications and visualizations
- `papers/`: Academic papers and manuscripts
- `results/`: Analysis outputs and datasets
- `sources/`: Source data and metadata
- `methods.md`: Methodology documentation
- `claims.md`: Research claims and hypotheses
- `data/README.md`: Data management and processing notes
- `manifest.json`: Project metadata and manifest
- `next.md`: Next steps and task planning

## Contribution Guidelines
- Follow the existing structure and naming conventions
- Document all changes and additions in `journal.md`
- Ensure all data sources are properly cited in `sources/sources.md`
