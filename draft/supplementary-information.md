## Supplementary Note 1. Caption-control register and denominator-discipline rules

**Source artifact:** `results/dashboard-paper/caption-control-register-v0.1.tsv`

**Purpose:** The caption-control register governs how every count, figure
number, denominator, and claim sentence travels from the live dashboard
exports into the manuscript. It prevents denominator drift, overread, and
stale caption reuse.

**Key rules:**
1. Every figure/table caption must name its denominator class explicitly.
2. No count may appear in prose without a corresponding authority file reference.
3. Dashboard panels and article figures must use identical denominator
   definitions; divergence requires a formal drift-audit entry.
4. Security, multilingual, public-sector, and provenance companions are
   always marked as bounded sidecars, never as core counts.
5. Translation-review burden must remain visible for multilingual rows.
6. Route C local-bundle provenance is separate from stronger witness states.

**Access:** The full register is available as a TSV export in the
reproducibility package under `results/dashboard-paper/`.
