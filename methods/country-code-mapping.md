# Country Code and Language Mapping

This document establishes standardized country codes and language mappings for the LIMEN AI Edge-Case Atlas project.

## Base Standard: ISO 3166-1 Alpha-2 Codes
| Code | Official Name | Common Name | Primary Languages |
|------|---------------|-------------|------------------|
| AD   | Principality of Andorra | Andorra | ca (Catalan) |
| AE   | United Arab Emirates | UAE | ar (Arabic) |
| AF   | Islamic Republic of Afghanistan | Afghanistan | fa (Persian), ps (Pashto) |
| AG   | Antigua and Barbuda | Antigua | en (English) |
| AL   | Republic of Albania | Albania | sq (Albanian) |
| DZ   | People's Democratic Republic of Algeria | Algeria | ar (Arabic) |
| AS   | American Samoa | American Samoa | en (English) |
| AD   | ... | ... | ... |

## Special Cases

### European Union (EU)
- Code: EU
- Note: Use for pan-EU institutions and regulations

### Kosovo
- Code: XKS (de facto state)
- Note: Use for references to Kosovo outside formal UN recognition
