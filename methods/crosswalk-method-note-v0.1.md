# Crosswalk Methodology Note v0.1

## Overview
This document describes the process used to create the `source-crosswalk-v0.1.tsv` mapping public AI incident/risk/vulnerability sources to permissible reuse terms.

## Research Steps
- Identified official public portals for each source.
- Extracted licensing information from project homepages and repository metadata.
- Summarized access permissions (e.g., academic research allowed, non‑commercial restrictions).
- Verified attribution requirements and any modification notices.
- Mapped each source to a standardized set of columns: `Source`, `AccessType`, `ReuseTerms`, `License`, `Notes`.
