## What the artifact adds

The facet converts the authoritative package into a reviewer-legible table that distinguishes:

1. direct regulator-hosted document text;
2. regulator public-record summaries without notice-text depth;
3. agency summary-only procedural chains.

Each row carries a bounded claim ceiling, reviewer use, dashboard role, paper-table role, exact upgrade need, and an explicit negative-result note where deeper document capture was not found.

## Paper-readiness delta

- Upgraded the authoritative package from a lane-local matrix feature into a reusable dashboard/paper input.
- Added a simple evidence-depth axis that a hostile reviewer can understand without re-reading four authority pages.
- Made the contrast between `LIMEN-000001` and the shallower EEOC/ICO rows explicit enough for methods, figure captions, and limitations text.
- Preserved conservative scope: the facet is about document depth and support ceiling, not legality, liability, or incident truth beyond the quoted authority surface.
