## Proof-ceiling discipline (per class)

Each class supports only bounded claims (see reviewed-core-evidence-panel TSV):
T3 supports the existence of a dated action; T2 supports only that a matter reached
a stage; CVE supports a documented vulnerability; media supports only that a
widely-reported event occurred. No class supports incident truth, prevalence,
legality-by-inference, or compliance.

## Method (gather -> adversarial verify -> tier -> bound -> link)

1. Gather from regulators, courts, prosecutors, reputable reporting.
2. Adversarial fact-check vs primary source (skeptic pass; drop/demote on doubt).
   Across verified batches, 0 fabrications were found; tier overstatements demoted.
3. Conservative evidence tier; contested/interim/recent flagged.
4. Per-case claim ceiling + explicit "what is not proven" caveat.
5. Every case linked to its primary source.
