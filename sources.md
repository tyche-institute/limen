# LIMEN Source Family Notes

## AIID (AI Incident Database)
- Description: Comprehensive database of AI-related incidents and failures
- Access: Public API with authentication
- Constraints: Non-commercial use, attribution required
- Scope: Global, English
- Update: Quarterly
- Cases: Long-tail incidents, system failures
- Saturation: Low

## OECD AIM (AI Policy Inventory)
- Description: OECD's collection of AI policy documents
- Access: Public website scraping
- Constraints: Creative Commons license
- Scope: Global, English
- Update: Annual
- Cases: Policy frameworks, normative gaps
- Saturation: Medium

## AVID (Adversarial Vision Incidents Database)
- Description: Database of adversarial attacks on vision systems
- Access: Public dataset download
- Constraints: MIT license
- Scope: Global, English
- Update: Semiannual
- Cases: Computer vision failures
- Saturation: Low

## MITRE ATLAS
- Description: Adversarial Threat Landscape for AI Systems
- Access: Public GitHub repository
- Constraints: MIT license
- Scope: Global, English
- Update: Biannual
- Cases: Attack patterns, safety issues
- Saturation: Medium

## CSET AI Harm
- Description: Stanford Center for Security and Emerging Technology AI risk cases
- Access: Academic reports
- Constraints: Creative Commons license
- Scope: Global, English
- Update: Annual
- Cases: Regulatory risks, cybersecurity
- Saturation: High
